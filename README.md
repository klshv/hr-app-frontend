# HrApp

HrApp is the client-side application for managing employee records within an organization. Built with Angular and TypeScript, this frontend interacts with the HR_app API. The UI components are styled using Bootstrap for a responsive and modern design.

## Features

- User-friendly interface for managing employee records
- Integration with HR_app API
- Responsive design using Bootstrap
- TypeScript client code generated by NSwag

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.2.6.
